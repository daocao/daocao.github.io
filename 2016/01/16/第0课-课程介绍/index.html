第0课 课程介绍

By 稻草大人

2016-01-16 更新日期:2016-01-24

第一节 这个课程学什么

学习内容提要：

  • 计算机基础知识（C语言相关）
  • C语言基础导学
  • C语言进阶导学
  • C语言项目实践

关于学习材料

  • C语言编程：一本全面的C语言入门教程（第三版）——教材

推荐书籍或资料

  • 代码大全（对Java程序猿也有帮助的书籍）
  • 高质量C语言编程
  • C语言库函数参考手册
  • C语言常见问题集

第二节 这个课程怎么学

  • 看视频
  • 结合视频看教材
  • 难点多理解
  • 动手多编程
  • 编码要规范

第三节 课程特色

  1. 真正的0基础课程，从计算机基础开始；
  2. 摒弃枯燥的C语言语法讲解，就将学习重点和难点，大量的实例；
  3. 真正有用的C语言课程，内含大量编程经验和技巧；
  4. 适合0基础，也适合进阶学习，内容由浅入深；
  5. 强调商业编码规范，以后求职C语言工作不是你的门槛；
  6. 老师实际编码操作，从程序设计需求分析，到流程图，到伪代码，到代码实现，真正告诉你该如何编程。

第四节 C语言的重要性

  1. 看看编程语言排行榜：2015编程语言排行榜前三：Java、C、C++；
  2. 现在那些开发还需要用C语言：单片机、 驱动开发（Linux）、IOS（OC）、PC；
  3. 那些人需要学习C语言；
  4. C语言中指针的存在，可以让我们直接操作寄存器地址，从而可以操作硬件，操作底层的能力非常强。

Tips:编程语言学习之路

  1. 学习语法；
  2. 编码规范；
  3. 程序设计的思维。
