第一节 为什么Python如此受欢迎

Python语言介绍：

  • 面向对象，解释型算计程序语言；
  • 1989年Guido van Rossom发明1991年公开版本发行；
  • 广泛应用于科学计算，众多开源学计算包提供ython接口；
  • 当今最受欢迎的编程语言之一，2004年后使用率线性增长；
  • 美国大学计算机编程入门课程最流行的语言（前十名计算机系中有8名使用Python）；
  • 设计哲学：优雅、明确、简单、可读性强；
  • 优点：功能强大、开发效率高、应用广泛、易上手、语法简洁。

Python语言的用途：

  • 网页开发
  • 可视化（GUI）界面开发
  • 网络
  • 系统编程
  • 数据分析
  • 机器学习
  • 网络爬虫
  • 科学计算

哪些著名的网站是Python实现的，例如：

  • 雅虎地图
  • 谷歌中的很多组成部分
  • Youtube
  • 豆瓣网
