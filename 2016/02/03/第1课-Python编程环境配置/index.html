第1课 Python编程环境配置1

发表于 2016-02-03   |   分类于 Python语言编程基础   |  

第一节 配置Python

　　使用Python语言开发，首先要安装Python的可执行文件，先要去Python的官网下载需要的安装包Python官网下载地址。　

　　下载Python完之后，点击next安装就可以了。

第二节 IDLE介绍

　　IDEL是Python安装装包自带的一个集成开发环境。是和Python一起安装的（前提是安装pyth时选中了“Tcl/Tk”组件），安装完Python的同时IDEL也安装完成了。

　　安装Python后，我们可以从“开始”菜单→“所有程序”→“Python”→“IDLE（Python GUI）”来启动IDLE，然后就用Python编写写程序了。

　　

第三节 PyDev介绍，配置Eclipse开发环境

　　PyDev是一个Eclipse针对Python开发的插件，在Eclipse里集成PyDev插件之后，我们就可以使用Eclipse编写Python程序了。下面是Mac版Eclipse配置PyDev的过程：

  1. 下载安装Eclipse：Eclipse for JavaEE下载地址

  2. 下载安装JDK1.6JDK1.6下载地址

  3. 下载PyDev：PyDev下载地址
  4. 解压PyDev，将解压后的文件夹里的plugins、features里的所有文件、文件夹，分别拷贝到Eclipse的plugins、features里。
  5. 配置PyDev，打开Eclipse打开Eclipse的偏好设置，点击右侧栏里的PyDev——>Interpreters——>PyDev Interpreter，点击右侧的按钮new，弹出对话框
  6. 在Interpreter Name随便填写个名字即可,Interpreter Executable 填写你python所在路径。

我下载的新的3.4，所以填：
/Library/Frameworks/Python.framework/Versions/3.4/bin/python3

  1. 最后，点[OK]，就可以创建Python项目了。

第四节 创建第一个Python项目

第一步，点击File ——> new ——> PyDev Project

弹出对话框：

第二步，在Project name随便填一个名字，作为项目名称。在Grammar Version选择与已经安装好的Python版本对应的版本号，例如：我的版本是3.4，则选中3.0。点击finish，项目创建完成。

第三步，右键点击项目名称，点击New ——> Pydev Module

弹出创建module对话框：

为module起一个名字，点击finish，一个module就创建完成了。

第四部，双击创建好的module，在编辑区输入代码，并运行项目。

print("Hello Python!!!")

运行后，控制台输出 Hello Python!!!，一个Python项目就创建成功了。
